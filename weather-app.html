Weather App

OVERVIEW

Weather App is a responsive, user-friendly tool that utilizes JavaScript to deliver real-time weather data for any location. By employing JavaScript for DOM manipulation, data retrieval from the Open Weather API and event listeners for user interaction, I ensured seamless interactivity and efficient updates.

This app dynamically responds to user input, offering a smooth experience on both desktop and mobile devices. It fetches weather data such as temperature, description, humidity, and wind speed, enhancing user engagement through animated transitions and visual representations.

TECHNOLOGIES

HTML, CSS, JavaScript, Git, Github, API, DOM Manipulation
