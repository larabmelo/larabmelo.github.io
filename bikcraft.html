Bikcraft Website

OVERVIEW

Bikcraft was my final project for the Origamid HTML & CSS Course, focusing on creating a responsive, user-friendly website from scratch. It prominently featured CSS Flexbox and Grid Layout for layout design, leveraged semantic HTML for modern structure, incorporated interactive forms, and used JavaScript to add dynamic animations, resulting in a polished and engaging web experience.

TECHNOLOGIES

HTML, CSS, JavaScript, Flexbox, Grid, Media Queries, NPM, Git, Github
DURATION June 2023 - July 2023
